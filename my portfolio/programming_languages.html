Programming Languages

  • 1. Python

    Python is a versatile and beginner-friendly language known for its simplicity and readability. It's widely used for web development, data analysis, artificial intelligence, and automation.

    Achievement: Ranked #1 in Python on HackerRank.

  • 2. SQL

    SQL (Structured Query Language) is used for managing and manipulating relational databases. It allows users to query and extract data, define and modify data structures, and perform various database operations.

    Achievement: Ranked #1 in SQL on HackerRank.

  • 3. C++

    C++ is a powerful and efficient programming language used for developing system software, game engines, operating systems, and high-performance applications. It provides low-level memory manipulation and supports object-oriented programming.

  • 4. Java

    Java is a popular programming language known for its portability, security, and robustness. It's commonly used for building enterprise-scale applications, web servers, Android mobile apps, and large-scale distributed systems.

  • 5. React.js

    React.js is a JavaScript library for building user interfaces, particularly for single-page applications. It allows developers to create reusable UI components and efficiently update the UI in response to data changes.

  • 6. Angular

    Angular is a TypeScript-based framework for building web applications and dynamic user interfaces. It provides a comprehensive solution for frontend development, including data binding, dependency injection, and routing.
